# Task 6 - Data Cleaning and Preprocessing

## Objective

Prepare a telecom churn dataset for machine learning by performing preprocessing operations.

## Steps Performed

- Loaded dataset
- Checked missing values
- Removed duplicates
- Encoded categorical variables
- Detected and treated outliers using IQR
- Performed feature engineering
- Standardized numerical features
- Saved cleaned dataset

- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-Learn
